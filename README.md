# -E-commerce-Store

A modern e-commerce application built with Django.

## Features
- User Authentication (Register/Login)
- Product Catalog
- Shopping Cart
- Order Management
- User Order History

## Setup
1. Clone the repository
2. Create a virtual environment: `python -m venv venv`
3. Activate the virtual environment
4. Install dependencies: `pip install -r requirements.txt` (Make sure to generate this if it doesn't exist)
5. Run migrations: `python manage.py migrate`
6. Start the server: `python manage.py runserver`
